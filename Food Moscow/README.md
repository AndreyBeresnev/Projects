Изучение рынка заведений общественного питания Москвы.

В проекте на основе открытых данных (8406 строк) с точками общепита в Москве исследовали рынок, чтобы определить наиболее перспективную нишу для открытия собственного заведения. 
Для этого сегментировали все заведения по категориям по территориальному расположению, по сетевым или несетевым заведениям, по количеству посадочных мест, по рейтингу и по среднему чеку.
Оценили с какими характеристиками наиболее коррелируют рейтинг заведений и средний чек.
Сделали рекомендации для открытия новой точки.

Структура проекта:

1.Загрузка данных и знакомство с ними.

2.Предобработка данных.

3.Исследовательский анализ данных.

4.Итоговые выводы.

Были использованы библиотеки pandas, numpy,  matplotlib, seaborn, phik.
