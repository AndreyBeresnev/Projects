Разработка A/B-тестирования и анализ результатов.

В проекте были исследованы исторические данные (435924 строк с сессиями пользователей) приложения с короткими видео. 
После были рассчитаны параметры A/B-тестирования нового алгоритма рекомендаций и оценена корректность проводимого эесперимента.
По результатам теста на 30579 пользователей ключевая метрика выросла на 3.4%, p-value = 0.00016, нововведение признано успешным.

Структура проекта:

1. Загрузка данных и знакомство с ними.

2. Подготовка к A/B-тесту.

3. Мониторинг A/B-теста.

4. Проверка результатов A/B-теста.

5. Итоговые выводы.

Были использованы библиотеки pandas, numpy, matplotlib, scipy, statsmodels.
