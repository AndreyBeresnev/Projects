Анализ системы метрик приложения.

В проекте я проводил бизнес-анализ приложения с развлекательным контентом. 
На основе данных 161864 пользовательских сессий, сопутствующих им покупок, а также маркетинговых затрат на привлечение клиентов из различных источников за 8 месяцев  я провёл когортный анализ клиентов, рассчитал CAC, LTV и ROI в разрезе каналов и когорт, определил причину стагнации выручки, дал характеристики для каждого канала привлечения и сделал рекомендации для отдела маркетинга.

Структура проекта:

1. Загрузка и предобработка данных.

2. Подготовка данных к когортному анализу.

3. Анализ месячной динамики основных метрик продукта.

4. Анализ метрик в разрезе источника привлечения.

5. Анализ периода окупаемости маркетинговых вложений.

6. Выводы и рекомендации.

Были использованы библиотеки pandas, seaborn, matplotlib.
